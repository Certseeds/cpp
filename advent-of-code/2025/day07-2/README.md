# day7-2

定义`dp[i][j]`为, 从初始点到(i,j)一共存在多少种可能性

对某一个点来说

1. 如果为初始点, 则为1
2. 如果这个点为EMPTY, 则显然为0
3. 如果这个点为一个分流器, 也显然为0
4. 如果这个点为快子流, 则需要进行判断

快子流的可能性来自其上方点, 左上方点, 右上方点.

+ 对于上方点, 直接判断上一行同点位是否存在快子流, 存在则 `dp[i][j]` += `dp[i-1][j]`
+ 对于左上方右上方点, 引入一个 `split_visited[i][j]`(0|1) 来表示这个分流器是否被快子流激发

则 `dp[i][j]` += `split_visited[i][j-1]` * `dp[i-1][j-1]` + `split_visited[i][j+1]` * `dp[i-1][j+1]`(注意对j约束, 不要越界)

最后对最后一行整个求和即可
